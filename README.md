# fliz.co.uk

Personal portfolio and technical experiments site. Built with Next.js 15, TypeScript, and Tailwind CSS. Hosted on Vercel.

## Quick Start

```bash
npm install
npm run dev
```

## Adding Experiments

1. Add entry to `src/lib/experiments.ts`
2. Create page at `src/app/experiments/[your-slug]/page.tsx`
3. Set status to `"live"` when ready

See [SETUP.md](./SETUP.md) for full deployment and DNS instructions.
